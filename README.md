# ResumeForge

ResumeForge is a Python and Flask resume builder inspired by the component structure of AI Song Maker: a tool-first hero, generation panel, saved output cards, workflow steps, testimonials, and FAQ sections.

## Features

- Login and register flow with per-user saved resumes
- Resume generator with role, skills, keywords, experience, and template controls
- Live resume preview with ATS score, summary, skills, and achievement bullets
- Save, boost, and delete generated resumes
- Mustard yellow-brown visual system with animated panels and cards

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open `http://127.0.0.1:5186/`.
